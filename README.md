# Install
## MacOS or Ubuntu
```shell
git clone https://github.com/qujihan/.dotfiles.git ~/.dotfiles && cd ~/.dotfiles
make download && make link
# macos only
./script/.macos --set
```

## Windows
```shell
scoop install sudo
git clone https://github.com/qujihan/.dotfiles.git ~/.dotfiles; cd ~/.dotfiles
make download_win; make link
```
